## ✨ Features

*   **⚡ High-Performance Rendering**: Built with Vite 8 for near-instant boot times and sub-second HMR.
*   **🌀 Interactive Background**: A physics-based Canvas network that reacts to mouse tracking and touch events with Repel/Attract mechanics.
*   **🛠️ Tech Terminal**: Integrated interactive terminal window with dynamic JSON payload rendering.
*   **📱 Progressive Web App (PWA)**: Fully installable on iOS, Android, and Desktop with custom brand icons and offline support.
*   **🛡️ Security Hardened**: Pre-audited for OWASP Top 10, featuring strict Content Security Policies (CSP) and HSTS.
*   **🖱️ Physics Cursor**: Custom dual-layer cursor with lagging physics for a premium cinematic experience.

## 📦 Installation & Setup

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/abhinaavchauhan/Portfolio.git
    cd Portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Start Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🔒 Security Posture

This project includes an **Enterprise-Grade Security Hardening** layer:
- **Strict Content Security Policy (CSP)**: Disallows `unsafe-eval` and unauthorized script injections.
- **HSTS Preloading**: Enforces HTTPS globally for 1 year.
- **Zero Source Map Leakage**: Production builds are stripped of `.map` files to prevent reconnaissance.
